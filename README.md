# Simple MVC Structure

## Writen in pure PHP
- Vanilla PHP
- No dependencies
- Except PHPMailer for email sending
- Easy to change if needed

## Features
- MVC
- Validation
- JWT + Blacklisting
- User profiles
- Email + Email Verification
- CSRF Protection
- Middleware
- MySQL Connection
- Logger
- Easy to load views and components
